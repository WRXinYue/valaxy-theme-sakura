## 侧边栏 SakuraSidebar <SupTag>0.6.0</SupTag>

配置侧边栏 (Sidebar) 和导航栏 (Navbar) 的方法大致相似。例如，NavItem 类型既可以用于侧边栏也可以用于导航栏。若需编写技术文档，您可选择使用 SidebarMulti 类型。需要注意的是，侧边栏提供了更多的配置选项

## API

### Slots

| 插槽名 | 说明 |
| --- | --- |
| default | 推荐放入`SakuraSidebar`主题组件 |
